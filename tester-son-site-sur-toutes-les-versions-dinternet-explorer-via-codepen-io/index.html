Astuce – Tester son code frontend dans toutes les versions d’Internet Explorer via CodePen.io

Découverte assez intéressante que j’ai faite aujourd’hui et qui permet de tester ses snippets html/css/js dans toutes les versions d’Internet Explorer. L’astuce est simple puisqu’elle utilise des VM (Machines Virtuelles) pour émuler les différentes versions d’IE. Pour résumer, on se connecte à son compte CodePen.io, on choisit son Pen, puis dans le menu à droite, en fin de liste, on a « Open on CrossBrowserTesting ».

Suite à ça on est redirigé vers le site app.crossbrowsertesting.com sur lequel on créé un compte gratuit puis on sélectionne la version du navigateur ainsi que l’OS voulu pour le test. C’est simple et efficace !
