Out of space !

L’un des problèmes majeurs que je rencontre avec Ubuntu, c’est que contrairement à Windows, non seulement il ne vous prévient pas quand votre disque sature, mais qu’en plus il peut cesser de fonctionner. Mais des remèdes existent !Depuis peu j’utilise essentiellement 2 utilitaires :

xdiskusage

Cet utilitaire permet de savoir rapidement quels sont les dossiers / fichiers sur votre volume qui prennent le plus de place. Il vous affiche la liste des volumes avec en vis à vis l’espace disque alloué et le pourcentage utilisé.

Pour savoir ensuite ce qui prend de la place sur le volume, il suffit d’un simple clic sur le volume souhaité pour qu’il affiche une nouvelle boîte de dialogue vous donnant un aperçu graphique de l’arborescence occupant le plus d’espace.

Il est ensuite tout à fait possible par double clic sur le dossier voulu, de descendre dans l’arborescence et observer d’où vient la fuite d’espace disque. Je trouve vraiment cet utilitaire parfait car il fait une seule chose et il la fait bien 🙂

Pour l’installer et le lancer :

1
2
3
4
 
sudo apt-get install xdiskusage
sudo xdiskusage
 

Ubuntu Tweak

Voici un autre utilitaire que j’ai découvert assez tard, bien qu’il soit assez célèbre. Les Ubunteros le connaissent bien, mais pour ceux qui ne le connaissent pas, je vous le présente ici.

En plus d’être très simple à prendre en main, cet utilitaire vous permet de vider tous les caches de votre système, de supprimer vos vieux kernels qui prennent de la place, ainsi que d’apporter quelques modifications pratiques à votre OS. Je vous laisse découvrir tout ceci par vous même.

Attention ! A l’heure où j’écris ces lignes, il n’existe pas pour les versions ultérieurs à Ubuntu 14.10. Cependant, il est tout à fait possible d’utiliser une version antérieures sur système 15.04.

Pour l’installer (version 14.10 et inférieures) :

1
2
3
4
5
 
sudo add-apt-repository ppa:tualatrix/next
sudo apt-get update
sudo apt-get install ubuntu-tweak
 

Pour les versions supérieures à 14.10 :

1
2
3
 
sudo add-apt-repository ppa:tualatrix/next
 

Editez ensuite le ppa en modifiant la version de l’os par la votre (remplacer vivid par votre version) :

1
2
3
 
sudo nano /etc/apt/sources.list.d/tualatrix-ubuntu-next-vivid.list
 

1
2
3
4
 
deb http://ppa.launchpad.net/tualatrix/next/ubuntu trusty main
# deb-src http://ppa.launchpad.net/tualatrix/next/ubuntu trusty main
 

devient :

1
2
3
4
 
deb http://ppa.launchpad.net/tualatrix/next/ubuntu vivid main
# deb-src http://ppa.launchpad.net/tualatrix/next/ubuntu vivid main
